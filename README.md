# UserConfigSaver
A [CounterStrikeSharp](https://github.com/roflmuffin/CounterStrikeSharp) plugin that saves CS2 player settings to MySQL when they spawn.

## Saves
| Category | Settings |
|----------|----------|
| HUD | `currently not available` |
| Viewmodel | `viewmodel_fov`, `viewmodel_offset_x/y/z` |
| Crosshair | Full share code |
| Mouse | `sensitivity` |
